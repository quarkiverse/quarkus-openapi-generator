== Installation

If you want to use this extension, you need to add the `io.quarkiverse.openapi.generator:quarkus-openapi-generator` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.openapi.generator</groupId>
    <artifactId>quarkus-openapi-generator</artifactId>
</dependency>
----
